# Autolayout-LAB
Autolayout programmatically 

## Autolayout의 핵심
- 해당 뷰의 X,y 위치 
- 해당 뷰의 가로, 세로 크기
 (양쪽에 CONSTRAINT 걸어져 있으면 Width를 잡아도 CONSTRAINT 충돌이 발생)
